## Shortcuts
### Text

| Shortcut | Description  |
| -------- | ------------ |
| `ctrl+c` | `copy`       |
| `ctrl+v` | `paste`      |
| `ctrl+f` | `find`       |

### Panes

| Shortcut    | Description |
| ----------- |------------ |
| `alt+Enter` | `Split (automatically)` |
| `alt-h`     | `Focus on the window on the left`                            |
| `alt-j`     | Focus down       |

### Tabs

| Hotkey       | Description     |
| ------------ | --------------- |
| `ctrl+t`     | `Duplicate tab` |
| `ctrl+w`     | `Close tab`     |
| `ctrl+alt+l` | `Next tab`      |
| `ctrl+alt+h` | `Prev tab`      |
