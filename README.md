## Shortcuts
### Text

| Shortcut | Description  |
| -------- | ------------ |
| `ctrl+c` | `copy`       |
| `ctrl+v` | `paste`      |
| `ctrl+f` | `find`       |

### Panes

| Shortcut      | Description   |
| ------------- |-------------- |
| `alt+enter`   | `split`       |
| `alt-h/j/k/l` | `focus <dir>` |

### Tabs

| Shortcut     | Description |
| ------------ |------------ |
| `ctrl+w`     | `close`     |
| `ctrl+t`     | `duplicate` |
| `ctrl+alt+l` | `next`      |
| `ctrl+alt+h` | `prev`      |
