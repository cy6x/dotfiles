## Shortcuts
### General

| Shortcut | Description  |
| -------- | ------------ |
| `ctrl+c` | `Copy text`  |
| `ctrl+v` | `Paste text` |
| `ctrl+f` | `Find text`      |

### Tabs

| Hotkey       | Description     |
| ------------ | --------------- |
| `ctrl+t`     | `Duplicate tab` |
| `ctrl+w`     | `Close tab`     |
| `ctrl+alt+l` | `Next tab`      |
| `ctrl+alt+h` | `Prev tab`      |

### Panes

| Shortcut                  | Description                          |
|---------------------------|--------------------------------------|
| `Alt+Enter`               | Split auto                            |
| `Alt-h`                   | Focus left                            |
| `Alt-j`                   | Focus down                            |
