## Shortcuts
### Text

| Shortcut | Description  |
| -------- | ------------ |
| `ctrl+c` | `copy`       |
| `ctrl+v` | `paste`      |
| `ctrl+f` | `find`       |

### Panes

| Shortcut      | Description   |
| ------------- |-------------- |
| `alt+o`       | `close`       |
| `alt+enter`   | `split`       |
| `alt+h/j/k/l` | `focus`       |
| `alt+shift+h/j/k/l` | `resize`      |

### Tabs

| Shortcut     | Description |
| ------------ |------------ |
| `ctrl+w`     | `close`     |
| `ctrl+t`     | `duplicate` |
| `ctrl+alt+l` | `next`      |
| `ctrl+alt+h` | `prev`      |
