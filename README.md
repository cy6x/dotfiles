## Shortcuts
### Text

| Shortcut | Description  |
| -------- | ------------ |
| `ctrl+c` | `copy`       |
| `ctrl+v` | `paste`      |
| `ctrl+f` | `find`       |

### Panes

| Shortcut      | Description  |
| ------------- |------------- |
| `alt+Enter`   | `split`      |
| `alt-h/j/k/l` | `focus left/down/up/right` |

### Tabs

| Hotkey       | Description     |
| ------------ | --------------- |
| `ctrl+t`     | `Duplicate tab` |
| `ctrl+w`     | `Close tab`     |
| `ctrl+alt+l` | `Next tab`      |
| `ctrl+alt+h` | `Prev tab`      |
